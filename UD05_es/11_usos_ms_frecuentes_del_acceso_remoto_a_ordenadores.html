1.1. Usos más frecuentes del acceso remoto a ordenadores

El software que permite la administración remota es cada vez más común y se utiliza a menudo cuando es difícil o muy difícil estar físicamente cerca de un sistema para usarlo o para acceder al material de Internet que no está disponible en la misma ubicación.

A continuación mostramos una lista de algunas de las tareas más comunes:

  • Generales: controlar un ordenador desde un lugar remoto. Por ejemplo: acceder a un ordenador personal desde un café de Internet. Apagar o reiniciar un ordenador remoto.
  • Acceder a periféricos Uso de un periférico conectado a la red, como por ejemplo una impresora. Recuperar datos de un flujo de datos, como por ejemplo de las cámaras de un circuito cerrado de televisión.
  • modificar datos Modificar la configuración de un ordenador remoto. Modificar el servicios que hay configurados en un servidor. Instalar o desinstalar software en un ordenador remoto.
  • visualizar Asistencia remota usuarios. Supervisar el uso de un ordenador o de la navegación por Internet.
