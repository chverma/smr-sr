Conceptos básicos de SSH

El objetivo de esta actividad es reforzar los conceptos tratados en el apartado sobre SSH del tema.

Responde en el documento sobre servicios de acceso y control remoto las siguientes cuestiones:

  1. Averigua qué afirmaciones de las que se muestra a continuación está erróneas y escribe de nuevo para que sean correctos:

    a) SSH, o lo que es lo mismo, Secure Shell, es una herramienta que permite realizar conexiones seguras y no seguras entre equipos conectados de forma segura como Internet.

    b) El servicio SSH utiliza el puerto 22 TCP y sigue el modelo cliente / servidor.

    c) Al utilizar este protocolo se consigue evitar el único riesgo importante, la interceptación de la información entre los equipos que se comunican.

  2. ¿Cuál dirías que es la función principal de SSH? ¿Qué otras funciones tiene?
  3. En el servicio SSH, como se autentica el usuario?
  4. ¿Qué problemas evita el uso del servicio SSH?
  5. Cuando es interesante utilizar el servicio SSH? Pon un ejemplo.
  6. ¿Qué diferencia existe entre un cliente SSH y un servidor? Pon un ejemplo.
  7. ¿Cuáles son las principales características del servicio SSH?
