Práctica 2: Instalación del servicio SSH

A. Acceso remoto mediante SSH

Objetivo

El objetivo de esta actividad es que os habituado al uso del cliente SSH tanto para acceder a equipos remotos como para transferir archivos.

SSH permite el acceso seguro a recursos remotos, ya sea en el sistema del equipo o en alguno de sus recursos.

Para hacer esta práctica deberá disponer de dos máquinas, la local y la remota. En caso de que no sea así, puede instalar el cliente y el servidor en la misma máquina.

Puede consultar el siguiente recurso:  https://help.ubuntu.com/16.04/serverguide/openssh-server.html

Recursos necesarios

Dos ordenadores, ya que un actuará de servidor SSH (al que nos conectaremos de forma remota), y el otro de cliente SSH, desde el que nos conectaremos. En su defecto podemos disponer de una máquina virtual que haga de servidor SSH.

El nodo que actúe de servidor SSH deberá disponer de un sistema operativo basado en Ubuntu.

Actividad

Siga las instrucciones para hacer la actividad:

  1. Instale el programa cliente y el servidor SSH: openssh-server.
  2. Acceda al terminal del equipo remoto.
  3. Qué pregunta aparecerá la primera vez que se conecte al equipo remoto? ¿Qué quiere decir?
  4. Lista el contenido de la carpeta en la que has accedido.
  5. Desde el cliente elige un archivo del equipo local y cópialo al equipo remoto mediante el mando scp . Explica los parámetros utilizados.
  6. Conectado mediante el protocolo sftp al servidor SSH. Descarga algún archivo.

Documenta el proceso.

B. Configuración del servicio SSH

Objetivo

El objetivo de esta actividad es que aprendáis a configurar los parámetros básico de un cliente y un servidor SSH.

Para hacer esta práctica deberá disponer de dos máquinas, la local y la remota. En caso de que no sea así, puede instalar el cliente y el servidor en la misma máquina.

Para obtener información de las opciones de configuración puede utilizar:

  • man ssh_config - para el cliente
  • man sshd_config - para el servidor

Actividades

Siga las instrucciones para hacer la actividad:

  • Configurar el cliente SSH [Linux Mint] de forma que al acceder al servidor indique el usuario por defecto y no sea necesario indicar al usuario.
  • Configurar el servidor de forma que sólo escucho por la dirección IP externa y sólo pueda acceder el usuario ftp_user . Configurar también que el tiempo de login no podrá superar los 60 segundos.

Documenta el proceso.

D. Acceso remoto desde Windows a Ubuntu con SSH

Objetivos

El objetivo de esta actividad es que aprendáis algunas utilidades para conectar equipos en Windows a servidores SSH basados ​​en Linux tanto en modo gráfico como en modo texto. 

Actividad. Cliente SSH por línea de mando

PuTTY es un cliente de terminal Transmission Control Protocol, desarrollado por Simon Tatham, distribuido como Software libre. Inicialmente fue desarrollado para funcionar sobre el Sistema Operativo Microsoft Windows pero actualmente existen versiones para varias plataformas Unix y Linux, y ya se encuentran en desarrollo versiones para Mac OS X.

Recursos: Equipo con Windows y equipo un sistema basado en Linux con el servidor OpenSSH instalado.

  1. Descárgate e instalado el servidor de ventanas Xming X Server (incluye el cliente SSH putty) en Windows. Más adelante utilizaremos el servidor de ventanas.
  2. Conectado al equipo con Linux
  3. Haz una captura de la conexión y otra de la sesión. 
información adicional

Windows 10 ya incluye de forma nativa un cliente y un servidor Open SSH, actualmente se encuentra en la versión beta. Si desea probarla ( Cómo activar el cliente OpenSSH oculto en Windows 10 )

Otras aplicaciones las podrás encontrar en Free SFTP, SCP and FTP cliente for Windows

Actividad. Conexión de forma gráfica desde Windows a un equipo con Linux.

Recursos: un equipo en Windows y el otro con Linux en entorno gráfico. 

  1. Instalado en Windows el servidor de ventanas xming (si no lo has hecho antes) y las fuentes xming-fuentes
  2. Haz una captura de la ejecución de alguna aplicación gráfica de Linux en Windows. 

Recurso de vídeo:  https://www.youtube.com/watch?v=QRsma2vkEQE

Recurso de texto:  http://courses.cms.caltech.edu/cs11/misc/xwindows.html
