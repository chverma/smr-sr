Activitat 4.1: FTP en mode consola

Prova les següents comandes del client FTP en mode consola:

  • help
  • ascii
  • binary
  • cd
  • chmod
  • close
  • quit
  • bye
  • delete
  • mkdir
  • get
  • dir
  • ls
  • put
  • pwd
  • prompt
  • rename
  • rmdir
  • size
  • passive
  • open

Hi ha també una sèrie de comandes interessants que comencen per "m" (ex: mput, mget, ...)

Després d'haver provat com funciona el client FTP en mode consola elabora una taula resum de les comandes que has utilitzat.

Activitat 4.2: Instal·lació i Configuració FTP anònim

Objectius

  • Aprendre a instal·lar i configurar un servei FTP en GNU/Linux.
  • Entendre el concepte d'usuari virtual.
  • Treballar amb els permisos en el sistemes de fitxers.

Activitats

  1. Instal·la i configura el servidor FTP (vsFTPd) perquè permeta l'accés anònim per a solament lectura a una carpeta anomenada public.
  2. Configura el DNS perquè tinga nom el servidor ftp. (ftp.##.aula.local).
  3. Afegeix alguns arxius a eixa carpeta i comprova el funcionament.
  4. Permet a l'usuari anònim penjar arxius en una nova carpeta anomenada incoming.
  5. Assegura't que els arxius es poden descarregar després de penjar-los.

Recursos

  • Instal·lació d'un servidor FTP en Ubuntu Server Guide: https://help.ubuntu.com/lts/serverguide/ftp-server.html
  • Informació de les opcions de configuració: https://security.appspot.com/vsftpd/vsftpd_conf.html 
  • Les opcions que afecten als usuaris anònims comencen per anon_
  • Tin en compte que l'usuari virtual necessita disposar d'un usuari local, en aquest cas és ftp.

Entregable

  1. Manual de passos realitzats per a la instal·lació i per a la configuració.
  2. Captures de pantalla que mostren el funcionament. Sols lectura, accés al directori, permisos, etc.

Activitat 4.3: Configuració VSFTP per a usuaris autenticats

Manual de vsftpd.conf

Recorda: Permisos en Linux

Configura el servidor FTP (vsFTPd)

  1. Modifica la taxa de transferència per als usuaris anònims a 10 bytes per segon.(anon_max_rate)
  2. Gestiona les connexions màximes per IP (2) [max_per_ip] i màxim número de clients (5) [max_clients].
  3. Ajusta els "timeouts" [timeout...]
  4. Ajusta el servidor per a permetre accés a alguns dels usuaris del sistema (no a tots) [local_enable, userlist_file, userlist_enable]. Crea un nou usuari ftp_user que serà l'únic que tindrà accés. [adduser]
  5. Document el procés, explicant els canvis realitzats.
