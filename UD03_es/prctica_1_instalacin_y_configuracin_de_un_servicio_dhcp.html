Práctica 1: Instalación y configuración de un servicio DHCP

objetivos

  • Instalar un servicio de configuración dinámica de los parámetros de red.
  • Presentar el servicio para asignar la configuración básica a los sistemas de una red local.
  • Realizar asignaciones dinámicas y estáticas.

recursos necesarios

Para poder realizar la práctica necesitarás:

  • Un ordenador con conexión a Internet o una máquina virtual con Ubuntu Server.
  • El software ISC DHCP.
  • La guía de instalación: https://help.ubuntu.com/16.04/serverguide/dhcp.html.ca https://help.ubuntu.com/community/isc-dhcp-server

enunciado

Para la red interna del entorno de trabajo instala el servicio DHCP y configúralo de forma que:

  • assign dinámicamente las 100 primeras direcciones disponibles
  • assign la configuración adicional necesaria para que los clientes DHCP puedan conectarse a Internet
  • los clientes resuelvan dinámicamente y con dominios relativos para el dominio de la red.
  • reserve la dirección IP 101 para el equipo con Windows XP.

Entregable

Guía de instalación y configuración del servicio DHCP en Ubuntu a partir de la Ubuntu Server Guide 4.16 (web) incluyendo:

  • Archivos de configuración básicos. Limitación porque el DCHP sólo funcione en la red interna. Cambios realizados explicando los efectos.
  • Archivo de concesiones (leases)
  • Registro del sistema donde se demuestre una concesión.
  • Captura del mando: sudo dhcpd -t -cf /etc/dhcp/dhcpd.conf
  • En Windows XP Ejecución de ipconfig / release Ejecución de ipconfig / renew Comprobación de la configuración asignada al Windows XP. Captura con wireshark del tráfico DHCP donde se compruebe todo el proceso.
