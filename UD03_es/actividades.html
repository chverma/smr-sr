4. ¿Qué indica cada línea del siguiente archivo de configuración de DHCP? Excepto 1) y 2).

autoritario;
identificador de servidor 192.168.1.1;
tiempo de arrendamiento predeterminado 86400;
tiempo máximo de arrendamiento 86400;
opción máscara de subred 255.255.255.0;
opción dirección de transmisión 192.168.1.255;
enrutadores opcionales 192.168.1.1;
subred 192.168.1.0 máscara de red 255.255.255.0
{
   rango 192.168.1.2 192.168.1.10;
   enrutadores opcionales 192.168.1.254;
}
host fantasia {
   hardware ethernet AA: BB: CC: DD: EE: FF;
   dirección fija 192.168.1.2;
}

5. Investiga que significan las líneas 1) y 2). Compruébalo menos en dos recursos.
