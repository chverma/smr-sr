5. Comandos en shell

Las siguientes utilidades son empleadas en la mayoría de sistemas operativos y permiten diagnosticar problemas, controlar los servicios y cambiar la configuración.

Ping

Ping (Packet Internet grupo) es una utilidad que sirve para enviar mensajes a una dirección concreta de red con el fin de realizar una comprobación utilizado el protocolo ICMP. El nodo destinatario nos responderá con un paquete para confirmarnos que los dos nodos pueden comunicarse perfectamente.

ifconfig

Permite obtener la información de la configuración IP del nodo, así como modificar de forma no persistente. El mando iwconfig permite ver la configuración de las conexiones inalámbricas.

netstat

netstat proporciona información sobre las conexiones de red.

ip route

ip route muestra la tabla de encaminamiento del nodo donde se ejecuta.

traceroute

traceroute nos muestra los enrutadores (routers) para que pasa un paquete hasta llegar a su destino.

nslookup

nslookup permite hacer consultas de nombres de dominio y comprobar qué servidor nos ofrece la respuesta.

ifup / ifdown

Las dos órdenes permiten activar o desactivar la interfaz de red indicada.

Para más información de cada utilidad puede usar la orden man seguido de la utilidad de la que se quiere mostrar información.

Por ejemplo: man nslookup

1.5.3. Registro del sistema (logs)


Todos los sistemas operativos incluyen un sistema para registrar los eventos que ocurren en el sistema operativo. En el caso de Linux estos registros son archivos de texto plano que se encuentran en la carpeta /var/log.


En la configuración y monitorización de los servicios de red tienen un papel muy importante porque nos pueden ayudar a diagnosticar los problemas que surgen.


Los servicios pueden tener su propio archivo de registro o usar el registro del sistema. El registro del sistema se guarda en el archivo /var/log/syslog y al tratarse de un archivo de texto lo podemos consultar simplemente usando el mando cat.




$ cat /var/log/syslog




Otra utilidad interesante es el comando tail, que permite visualizar las últimas líneas de los archivos de texto. Con el argumento -f muestra las últimas líneas en tiempo real, es decir, a medida que va modificándose el archivo. Mediante el argumento -n podemos indicar el número de líneas que es posible visualizar. En el siguiente ejemplo se muestra en tiempo real las últimas 20 líneas del archivo de registro del sistema.




$ tail -fn 20 /var/log/syslog
