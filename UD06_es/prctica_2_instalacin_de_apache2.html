Práctica 2: Instalación de Apache2

06.02: Instalación de Apache2

objetivo

  • Aprender a instalar Apache2
  • Conocer sus archivos más importantes
  • Configurar el servidor web por defecto.
  • Verificar los archivos de registro.

actividad

  1. Instala el servidor web Apache2. 
  2. Comprueba que puedes acceder tanto desde la red interna como la externa.
  3. Donde se guarda la configuración del servidor virtual predeterminado? Y la configuración general de Apache2?
  4. ¿Cuál es la ruta del directorio desde el que se han de utilizar los archivos del servidor virtual predeterminado?
  5. Donde se encuentra el archivo httpd.conf?
  6. ¿Qué diferencia hay entre el directorio conf-availabley conf-enabled?
  7. En qué archivo se encuentra la configuración del sitio web virtual por defecto?
  8. Configurar el DNS de forma que la dirección: www.aula ##. Lan permita el acceso al servidor web, tanto desde la red interna como desde la externa.

recursos

En la Ubuntu Server Guide dispones de la información necesaria para hacer la instalación, la configuración y responder las preguntas.
