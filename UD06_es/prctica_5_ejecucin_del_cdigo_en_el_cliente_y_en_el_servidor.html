Práctica 5: Ejecución del código en el cliente y en el servidor

06.05: Execució de codi. Client i Servidor

objetivo

actividad

Crea la siguiente página web y guarda con el nombre: default.php

<html>
<body>
<h1>Data</h1>
<?php
  echo "<h2>Servidor (php)</h2>";
  echo date("r") ;
?>
<script type="text/javascript">
  var d = new Date();
  document.write("<h2>Client (javascript)</h2>"+ d.toString());
</script>
</body>
</html>

Accede a la página mediante la URL: http://www.00.aula.local/default.php

¿Qué ha pasado? ¿Por qué?

Instala el intérprete PHP y el módulo de PHP para Apache2:

sudo apt install php libapache2-mod-php

Vuelve a acceder a la página http://www.00.aula.local/default.php. ¿Qué ha cambiado? Comprueba la diferencia con la ejecución anterior.
