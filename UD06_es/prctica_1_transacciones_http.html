Práctica 1: Transacciones HTTP

1.6: Mandos HTTP 1.1

objetivo

Entender cómo funciona el intercambio de mensajes HTTP.

funcionamiento

Conectamos con el servidor http mediante telnet.

$ telnet XXX.XX.XX.XXX 80

Hacemos la solicitud HTTP utilizando el encabezado deseado.

GET / HTTP/1.1 
host: www.example.com
User-Agent: Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 5.1) [opcional]
[INTRO]
[INTRO]

En el ejemplo anterior solicitamos mediante el encabezado GET el documento raíz (/) con HTTP / 1.1 y el host: www.example.com

actividad

Haz las pruebas con Telnet del mando HEAD en las siguientes URL:

  • http://www.pego.org (resuelto)
  • http://www.marca.com
  • http://moodle.iesevalorpego.org
  • http://forbes.es/demo
  • http://revistadefilosofia.com/60-05.pdf
  • http://www.fincarustica.cat/contacto/lang/es

Elabora un documento con capturas de pantalla que demuestran el funcionamiento y describe los problemas en que te has encontrado y las soluciones adoptadas. Anota también los código de respuesta y el tipo de archivo vuelto.

ejemplo

Para la primera URL haríamos:

$ telnet www.google.es 80

Una vez realizada la conexión hacemos la solicitud

HEAD / HTTP/1.1
host:www.google.es
[intro]
[intro]

Y obtendremos la siguiente respuesta:

HTTP/1.1 200 OK
Date: Fri, 12 Jan 2018 09:28:23 GMT
Expires: -1
Cache-Control: private, max-age=0
Content-Type: text/html; charset=ISO-8859-1
[contenido omitido]
